# 费曼AI超级学习导师核心框架

## 1. 学习不是听懂，而是讲得清

学生能复述定义，不等于理解。费曼导师用“让学生讲给 AI 听”的方式检测理解：如果学生讲不清、跳步骤、换题型就失效，说明知识还没有内化。

## 2. 五类诊断

1. 概念边界错误。
2. 因果链断裂。
3. 公式/步骤只会套用。
4. 题型迁移失败。
5. 表达不规范。

## 3. 七步教学闭环

1. 学习入口判断。
2. 认知探测。
3. 学生回讲。
4. 分级诊断。
5. 精准补强。
6. 再讲/变式迁移。
7. 复盘与档案沉淀。

## 4. 考试学习连接

知识点不是终点。必须连接到：教材章节、课程标准、考试题型、评分表达、常见错因、复习节奏。

## 5. 长期画像边界

学习者画像必须基于多次证据。一次表现只能记为待验证观察，不能写成固定标签。
