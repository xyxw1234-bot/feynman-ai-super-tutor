# 中国小初高官方课程教材资源图谱（v1.3.3）

## 已核查的公开来源

### 1. 国家中小学智慧教育平台

- 平台首页：`https://basic.smartedu.cn/`
- 课程教学频道：`https://basic.smartedu.cn/syncClassroom/auto`
  - 页面结构可见：学段、年级、学科、教材版本、册次；示例默认展示“小学 · 一年级 · 语文 · 统编版 · 上册”。
  - 用途：同步课程资源、课时资源、课程教学入口。
- 教材/电子教材入口：`https://basic.smartedu.cn/elecEdu`
  - 页面标题/路径：电子教材；页面筛选可见“小学、初中、小学（五四学制）、初中（五四学制）、高中、特殊教育”。
  - 例示筛选：学科、版本、年级；默认可见“道德与法治、语文、数学、英语、艺术·音乐、艺术·美术、科学、语文·书法练习指导、体育与健康、信息科技”等学科，以及统编版、一年级等。
  - 平台前端导航也存在“教材”频道路径。
- 合规用法：做索引、链接、章节定位、学习建议、少量摘要和学生学习路径；不批量复制、重打包或内置教材全文。

### 2. 教育部义务教育课程方案和课程标准（2022年版）

- 通知页：`http://www.moe.gov.cn/srcsite/A26/s8001/202204/t20220420_619921.html`
- 信息：教育部印发《义务教育课程方案和课程标准（2022年版）》，2022年秋季学期开始执行。
- 附件包含：义务教育课程方案，以及道德与法治、语文、历史、数学、英语、地理、科学、物理、化学、生物学、信息科技、体育与健康、艺术、劳动等课程标准。
- 用途：提取核心素养、课程内容、学业质量、教学提示和评价要求，转为学习目标、考点能力、费曼追问和练习设计依据。

### 3. 普通高中课程方案和课程标准（2017年版2020年修订）

- 通知页：`http://www.moe.gov.cn/srcsite/A26/s8001/202006/t20200603_462199.html`
- 信息：教育部印发普通高中课程方案和语文等学科课程标准（2017年版2020年修订），附件为课程方案和各学科标准打包文件。
- 用途：高中阶段课程结构、核心素养、学业质量、模块内容和评价要求。

### 4. 教育部课程教材研究所/基础教育课程教材发展中心

- 义务教育课程标准下载索引：`https://www.ictr.edu.cn/download_center/ywjy.html`
- 普通高中课程标准下载索引：`https://www.ictr.edu.cn/download_center/put.html`
- 用途：作为课程标准下载和版本核对的辅助索引。

## Skill 内部使用原则

1. 用户给出具体章节/教材/知识点时，优先形成“定位卡”：学段、学科、教材版本、册次、章节、课程标准领域、核心能力、考点、易错点、训练建议。
2. 未确认教材版本时，可以先给通用学习路径，但必须提示后续可用封面/目录校准。
3. 官方真题/样题必须核验来源、年份、地区和链接；否则只能称“原创模拟题/考试风格变式”。
4. 平台资源公开可合法用于学习索引和链接推荐，但不得批量搬运教材全文或题库全文。
