# 更新日志

## v1.1.0

- 发布“可视化互动学习增强版”。
- `SKILL.md` 增加视觉互动增强协议：判断何时需要图示、配图、互动 H5 或继续文字追问。
- 插件新增视觉必要性判断、H5 需求说明、互动 H5 生成、视觉素材静态质检和素材列表工具。
- 内置一次函数、二次函数、浮力、受力分析、电路、化学粒子等高频互动模板。
- 强化质量红线：H5 发给学生前必须完成静态检查、浏览器预览、移动端宽度检查、学科正确性复核和公网链接验证。
- 更新 README、INSTALL、验收脚本和可视化互动参考文档。
