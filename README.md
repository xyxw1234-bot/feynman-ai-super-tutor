# 费曼AI超级学习导师

面向 Hermes Agent〔智能体框架〕、节点引擎飞书智能体和其他 LLM Agent〔大语言模型智能体〕的可安装学习导师能力包。它把智能体从“答题机器”升级为“长期费曼学伴”：让学生先讲，智能体追问定位盲点，补强关键一步，并沉淀学习档案与复习计划。

## 适合谁

- 学生：学新知识、讲错题、备考复习。
- 家长：希望孩子不是只抄答案，而是真懂。
- 教师：把材料、题目、讲义转成可追问的学习路线。
- 节点引擎/Hermes 用户：希望自己的飞书智能体变成长期学习导师。

## 直接安装入口

把下面 raw 链接发给支持 Skill 安装的智能体，让它读取并安装：

```text
https://raw.githubusercontent.com/OWNER/feynman-ai-super-tutor/main/SKILL.md
```

发布到你的 GitHub 账号后，请把 `OWNER` 替换为实际账号名。

## 核心能力

1. 角色反转：学生先讲，AI〔人工智能〕不先灌答案。
2. 认知边界定位：用少量追问找“刚好差一步”的地方。
3. 分级诊断：关键误解、理解不完整、表达可优化。
4. 材料学习：把合法资料转为话题地图和学习路径。
5. 考试适配：连接知识点、考点、题型、易错点和规范表达。
6. 长期档案：保存主题卡、错因卡、表达卡、迁移卡和学习者画像。

## 可选 Hermes 插件

仓库内提供 `plugins/feynman_super_tutor/`，用于在 Hermes profile〔配置档案〕内本地保存学习档案、生成复习计划、将材料转为话题地图。插件不依赖外部服务器，不创建外部数据库，数据默认保存在当前用户自己的 Hermes profile 数据目录。

如果用户环境没有安装插件，`SKILL.md` 仍可独立运行；只是学习档案会依赖智能体自身的记忆或会话。

## 正版资源边界

本项目不内置盗版教材、盗版教辅或付费题库全文。它只处理：

- 用户合法输入或上传的资料；
- 官方公开资源；
- 学校/教师/家长有权提供的材料；
- 已获授权的正版教辅资源。

后续可以接入国家中小学智慧教育平台、人教社公开资源、出版社授权资源、学校自有资料库，但必须遵守版权和未成年人数据保护边界。

## 示例用法

```text
我想学初中数学一次函数，你先测我哪里不会。
```

```text
这道题我先讲我的思路，你帮我找漏洞。
```

```text
我把教材页发给你，你别直接总结，帮我变成可提分的学习路线。
```

```text
按费曼法复习今天学过的浮力，先问我三个问题。
```
